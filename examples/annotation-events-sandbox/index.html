    Annotation Events Comparison

    This example demonstrates both the official PDF.js Annotation Events API and the legacy custom events.

    Official API Events

    The official Annotation Events API provides a standardized way to interact with PDF annotations through the annotationEventManager:

    • Accessed via pdfDocument.annotationEventManager
    • Provides typed events with consistent data structures
    • Includes methods for managing annotations programmatically

    Legacy Custom Events

    The legacy events use custom DOM events with the pattern pdfjs-annotations-*:

    • Dispatched directly to the document object
    • Data is passed through the detail property
    • Some events use postMessage for communication
